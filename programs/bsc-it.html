Curriculum Structure

Comprehensive 4-year program with progressive skill building

Year 1: Foundation Building

Focus: Programming fundamentals, mathematics, and basic computer science concepts

Semester 1

  • Introduction to Programming (Python)
  • Computer Fundamentals
  • Mathematics for IT
  • Digital Logic Design
  • Communication Skills
  • IT Ethics & Society

Semester 2

  • Object-Oriented Programming (Java)
  • Data Structures & Algorithms
  • Statistics for IT
  • Computer Architecture
  • Web Development Basics
  • Project Management Fundamentals

Year 2: Core Technologies

Focus: Database systems, networking, advanced programming, and system analysis

Semester 3

  • Database Management Systems
  • Computer Networks
  • Software Engineering
  • Operating Systems
  • Advanced Web Development
  • Human-Computer Interaction

Semester 4

  • Advanced Database Systems
  • Network Security
  • System Analysis & Design

  • Internship Preparation

Admission Requirements

Join our competitive B.Sc IT program and start your technology journey

Eligibility Criteria

  • Academic: +2 completed in Science/Management with minimum 60% marks
  • Age: 17-25 years at the time of admission
  • English: IELTS 6.0+ or equivalent (international students)
  • Mathematics: Strong foundation in mathematics recommended
  • Programming: Prior programming experience preferred but not mandatory

Selection Process

  • Online application submission
  • Entrance examination (Math, English, Logical Reasoning)
  • Personal interview
  • Portfolio review (if applicable)

Application Timeline

Phase Date Activity
Phase 1 Jan 15 - Mar 30 Application Submission
Phase 2 April 5 Entrance Examination
Phase 3 April 10-15 Personal Interviews
Phase 4 April 20 Results Announcement
Phase 5 May 1-15 Admission Confirmation

Required Documents

  • +2 transcripts and certificates
  • Passport-size photographs
  • Citizenship certificate/passport
  • Character certificate
  • Medical certificate
