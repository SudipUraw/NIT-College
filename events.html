Event Registration

How to register and participate in NIT events

Registration Process

  1. Browse events and select the ones you want to attend
  2. Click "Register" button on the event card
  3. Fill out the registration form with required information
  4. Pay registration fee if applicable
  5. Receive confirmation email with event details
  6. Attend the event and enjoy the experience!

Registration Benefits

  • Guaranteed seat reservation
  • Event materials and resources
  • Certificate of participation
  • Networking opportunities
  • Priority booking for future events

Event Policies

Attendance

  • Arrive 15 minutes before event start time
  • Bring valid student/staff ID for verification
  • Follow event-specific dress codes if mentioned

Cancellation

  • Cancel up to 48 hours before event for full refund
  • 24-48 hours: 50% refund available
  • Less than 24 hours: No refund but credit for future events
  • Medical emergencies: Full refund with documentation

Special Requirements

  • Inform us of any accessibility needs
  • Dietary restrictions for catered events
  • Technical requirements for workshops
